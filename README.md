
# **Weather-Whiz**

Weather-Whiz is a full-stack web application that predicts weather conditions and displays nearby tourist attractions based on user input. It also features an interactive world map where users can click any location to instantly view live weather updates. The UI follows a pastel-themed, responsive design for a smooth and visually appealing experience.

---

## **Features**

* Real-time weather forecasting using external APIs
* Tourist attraction suggestions based on location
* Interactive map (click anywhere to get weather details)
* Pastel-themed, aesthetic, and responsive UI
* Smooth navigation, hover effects, and user-friendly layout

---

## **Technologies Used**

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Django

**APIs Integrated:**

* WeatherAPI
* OpenCage Geocoding
* Foursquare Places API

**Mapping:**

* Leaflet.js
